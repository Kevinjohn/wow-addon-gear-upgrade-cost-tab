# Changelog

What changed in each release, in plain language. Newest first.

<!-- Developer-level detail (APIs, data sources, rejected approaches) lives in CHANGELOG-dev.md. -->

## 0.10.1 — 2026-06-25
- Added a screenshot to the project page and documented the **Show fully upgraded gear** option in the README. No in-game changes.

## 0.10.0 — 2026-06-25
- Added **Show fully upgraded gear** (on by default): untick it to hide equipped gear you can't upgrade right now — both fully upgraded items and gear with no upgrade track — so the list shows only what crests or gold can still improve.

## 0.9.1 — 2026-06-25
- New releases now publish to CurseForge automatically.

## 0.9.0 — 2026-06-25
- Out of alpha.

## 0.8.1 — 2026-06-23
- Release and packaging only — no in-game changes.

## 0.8.0 — 2026-06-22
- Updated for retail patch 12.0.7, so the addon no longer shows as out of date.

## 0.7.0-alpha — 2026-06-11
- Added **Show my crests** (off by default): a footer showing how many of each crest your warband owns.

## 0.6.0-alpha — 2026-06-11
- Added a filter to show **Warbound until equipped** items in the bag lists (off by default — upgrading one binds it to you).

## 0.5.0-alpha — 2026-06-10
- Added bag filters to include **Uncommon** and **Rare** items (off by default).
- Added **Prioritise tier** (on by default): hides bag items for any slot where you're already upgrading a tier/set piece.
- Empty lists now tell you when items are hidden by your filters, instead of saying there's nothing to upgrade.

## 0.4.0-alpha — 2026-06-10
- Added Korean and Chinese (Simplified and Traditional) — every WoW language is now supported.

## 0.3.0-alpha — 2026-06-10
- Added German, French, Spanish, Italian, Portuguese, and Russian.

## 0.2.0-alpha — 2026-06-10
- Added two bag lists: items whose next upgrade is **free**, and items that **need crests**.

## 0.1.0-alpha — 2026-06-10
- First release: a **Gear Upgrades** tab on the character screen showing each equipped item's level, upgrade track, and crest cost to the next rank and to fully upgraded — with base or discount-aware pricing.
