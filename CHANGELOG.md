# Changelog

What changed in each release. Newest first.

<!-- Developer-level detail (APIs, data sources, rejected approaches) lives in CHANGELOG-dev.md. -->

## 0.10.1 — 2026-06-25
- Added a screenshot to the project page.
- Documented **Show fully upgraded gear** in the README.
- No in-game changes.

## 0.10.0 — 2026-06-25
- Added **Show fully upgraded gear** (on by default).
- Untick it to hide gear you can't upgrade — fully upgraded items, and gear with no upgrade track.

## 0.9.1 — 2026-06-25
- Releases now publish to CurseForge automatically.

## 0.9.0 — 2026-06-25
- Out of alpha.

## 0.8.1 — 2026-06-23
- Packaging only. No in-game changes.

## 0.8.0 — 2026-06-22
- Updated for patch 12.0.7. No longer shows as out of date.

## 0.7.0-alpha — 2026-06-11
- Added **Show my crests** (off by default).
- Shows how many of each crest your warband owns, in a footer.

## 0.6.0-alpha — 2026-06-11
- Added **Include Warbound items** (off by default).
- Off by default because upgrading one binds it to you.

## 0.5.0-alpha — 2026-06-10
- Added **Include Uncommon** and **Include Rare** bag filters (off by default).
- Added **Prioritise tier** (on by default) — hides bag items for slots where you wear a tier piece.
- Empty lists now say when filters are hiding items.

## 0.4.0-alpha — 2026-06-10
- Added Korean and Chinese (Simplified and Traditional).
- Every WoW language is now supported.

## 0.3.0-alpha — 2026-06-10
- Added German, French, Spanish, Italian, Portuguese, and Russian.

## 0.2.0-alpha — 2026-06-10
- Added two bag lists: next upgrade **free**, and **needs crests**.

## 0.1.0-alpha — 2026-06-10
- First release: a **Gear Upgrades** tab on the character screen.
- Shows each item's level, upgrade track, and crest cost to the next rank and to full.
- Base or discount-aware pricing.
